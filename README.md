# Mini Instagram Clone
A simple Instagram-like web application built using HTML, CSS, and JavaScript.  
This project was created with the help of GitHub Copilot to explore AI-assisted development and practice building interactive web apps.

## Project Description

Mini Instagram Clone simulates the core features of a social media feed:
- Scrollable feed with multiple posts
- Each post displays a user avatar, username, image, caption
- Like button with a dynamic counter
- Comment section where users can type and submit comments
- Responsive design for desktop and mobile

The goal of this project is to demonstrate building an interactive web interface entirely in the front-end, using AI to accelerate development.

## Features

- Fixed header with app name
- Scrollable feed with multiple posts
- Like button increments likes dynamically
- Comment input under each post displays submitted comments
- Responsive layout using CSS for mobile and desktop
- Basic styling with hover effects for interactivity

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/mini-instagram.git
   cd mini-instagram
````

2. Open `index.html` in your preferred web browser.

No additional dependencies or server setup is required since this is a front-end only project.
